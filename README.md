## How to Run the Program

### 1. Open a Terminal

- **Windows (CMD):**  
  1. Press `Windows + R`  
  2. Type `cmd` and press **Enter**

- **Linux:**  
  - Press `Ctrl + Alt + T`

- **macOS:**  
  1. Press `Command + Space`  
  2. Type `terminal` and press **Enter**

---

### 2. Verify if g++ is Installed

Run in terminal (works on Windows, Linux, macOS):  
```bash
g++ --version
```
